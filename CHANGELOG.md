# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2025-12-18

### Added
- Initial release of CeloSwap Mini
- Token swapping interface for Celo blockchain
- Support for CELO, cUSD, cEUR, cREAL, and USDC tokens
- Wallet integration with MetaMask and WalletConnect
- Real-time balance display
- Approximate USD equivalent calculations
- Transaction history view
- Price chart visualization
- Error boundary for better error handling
- Loading states and components
- Responsive design with Tailwind CSS
- TypeScript support
- GitHub Actions CI workflow
- Comprehensive documentation
- Test setup with Jest

### Features
- Lightning-fast token swaps
- Customizable slippage tolerance
- Real-time quote updates
- Wallet balance tracking
- Transaction monitoring

### Technical
- Built with Next.js 14 and React 18
- Wagmi for blockchain interactions
- RainbowKit for wallet connections
- Viem for Ethereum interactions
- Tailwind CSS for styling

### Known Issues
- Contract needs to be deployed on mainnet with liquidity
- Exchange rates need to be set manually in contract
